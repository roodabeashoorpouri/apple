# Online Stationary Store

This is a React project for an online stationary store, inspired by the Yoreska store in Taiwan. The project is based on the Comfy Sloth project by John Smilga, which provides a good starting point for building an e-commerce website.

## Installation

To run the project locally, clone the repository and install the dependencies:

```
git clone https://github.com/your-username/online-stationary-store.git
cd online-stationary-store
npm install
```

## Usage

To start the development server, run the following command:

```
npm start
```

This will start the server and open the website in your default browser at [http://localhost:3000](http://localhost:3000).

## Features

The online stationary store project includes the following features:

- A home page that displays featured products and categories
- A product detail page that shows more information about a specific product
- A cart page that displays the items in the user's shopping cart and allows them to update quantities and remove items
- A checkout page that collects the user's shipping and payment information and submits the order
- Authentication and authorization using Firebase
- Integration with the Stripe API for processing payments
